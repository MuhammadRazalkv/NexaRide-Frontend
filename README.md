# NexaRide 🚖  
A modern and responsive taxi service application built using **React, TypeScript, TailwindCSS, and Framer Motion**.  



---

## 🚀 **Features**
- 🚗 **User Authentication** (Login, Signup, OTP Verification)
- 🎨 **Responsive Design** (Mobile-first UI)
- 🔄 **Smooth Animations** (Framer Motion)
- 🌍 **Map Integration** (Geoapify)
- ⚡ **Fast & Optimized Performance** (Vite)

---

## 🛠️ **Tech Stack**
- **Frontend:** React (TypeScript), TailwindCSS, Framer Motion
- **State Management:** React Context API / Redux
- **API Handling:** Axios / Fetch API  
- **Authentication:** Firebase Google Auth
